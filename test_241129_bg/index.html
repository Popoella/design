이미지의 크기 조절

이미지 태그로 코딩한 이미지는 넓이나 높이 하나만 지정해주면 비율게 맞게 리사이즈됨 (이미지 크기 조절할 때 넓이/높이 동시에 주면 안됨)
이미지의 크기를 px또는 % 등으로 조정할 수 있는데, 이미지는 감싸는 요소의 넓이를 기준으로 계산함(원래 이미지 크기와 관계 없음)

object fit - contain

메인페이지 최근게시물에 이미지가 있는 경우, 담당자가 어떤 이미지를 업로드 하던 디자인 한 영역에 이미지를 찌그러지지 않게 구현하는게 중요함
contain-영역안에 이미지가 잘리지 않고 다 들어감

background-size:contain

배경을 주는 요소에 배경이미지를 짤리지 않게 모두 표시하는 방식, 배경의 일부에는 배경이미지가 들어가지 않음(빈 곳이 발생) 단, 이미지가 잘릴 수도 있음

  • 글자 글자

background-size:직접입력

배경의 크기를 px또는 %로 직접 입력 할 수도 있음. 이때는 img태그와 동일하게 넓이/높이 하나만 입력하면 비율에 맞게 사이즈가 줄어듦(절대 동시에 주면 안됨)
background-size: 100px; - 넓이가 조절됨 (background-size: 100px auto; 가 정확한 표현)
background-size: auto 100px; - 높이가 100px로 조정됨
만약 단위를 %로 주면 배경을 준 요소의 넓이를 기준으로 %계산(배경이미지 사이즈 관계 없음)
